# BCP - Biblioteca Carvalho Pereira

![Badge em Desenvolvimento](http://img.shields.io/static/v1?label=STATUS&message=EM%20DESENVOLVIMENTO&color=GREEN&style=for-the-badge)

## 📋 Sobre o Projeto

O **BCP** é uma aplicação web desenvolvida em **.NET (C#)** e **Blazor**, projetada para gerir os dados dos catálogos de livros em uma biblioteca residêncial. 
O sistema foca em oferecer uma interface simples e intuitiva com funcionalidades essenciais para administrar um pequeno espaço de biblioteca.

## 🚀 Funcionalidades Principais

* **Gestão de Cadastros (CRUD):** Criação, leitura, atualização e exclusão de registos.

## 🛠️ Tecnologias Utilizadas

O projeto está a ser desenvolvido utilizando as seguintes tecnologias:

* **C# 10**
* **Blazor**
* **.NET 8**
* **Entity Framework**
* **HTML5 & CSS3**
* **SQLite**
